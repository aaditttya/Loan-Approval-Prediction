# 📖 Project Overview

LoanWise AI is a Machine Learning powered web application that predicts whether a loan application is likely to be approved based on applicant information.

The project combines Machine Learning, Flask, SQLite, authentication, analytics dashboards, and a modern responsive user interface into a complete production-ready application suitable for portfolios and academic projects.

This project demonstrates the complete workflow of a real-world Machine Learning application—from data preprocessing and model training to deployment and dashboard analytics.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/aaditttya/Loan-Approval-Prediction.git
```

## Navigate into Project

```bash
cd Loan-Approval-Prediction
```

## Create Virtual Environment

```bash
python -m venv venv
```

## Activate Environment

### Windows

```bash
venv\Scripts\activate
```

### macOS/Linux

```bash
source venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Application

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```
